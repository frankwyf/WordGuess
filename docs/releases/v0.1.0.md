# WordGuess v0.1.0

## Highlights

- First public open-source release of WordGuess as a portfolio-ready Java terminal game.
- Daily play mode, fixed puzzle mode, and accessibility mode are all available from the Gradle tasks.
- Standard release artifacts are included, plus a Windows-native bundle for double-click launch.

## What Is Included

- Java 26 + Gradle 9.4 build setup
- JUnit test suite and Checkstyle validation
- GitHub Actions CI for build, test, and packaging
- Automated GitHub Release pipeline for version tags
- Open-source project files for licensing, contributing, and security reporting

## Recommended Downloads

- `WordGuess-windows.zip`: recommended for Windows users who want to extract and double-click `WordGuess.exe`
- `wordguess-all.jar`: best option if you want a single-file runnable build
- `wordguess.zip`: best option if you want the standard packaged app layout generated by Gradle

## Run Locally

```powershell
.\gradlew.bat run
```

Run a fixed puzzle for demo purposes:

```powershell
.\gradlew.bat runFixed
```

## Notes

- This is the first public release, so the focus is on a clean build, playable core loop, and release automation.
- Future releases can expand gameplay polish, terminal UX, and release assets.
